# My To-Do List App ^__^

A simple command-line To-Do List application made with Python.

## Features

* Add new tasks
* View all tasks
* Mark tasks as completed
* Mark tasks as priority
* Delete tasks
* Error handling for invalid input

## Technologies Used

* Python 3.11.9
* Visual Studio Code

## How to Run the Program

1. Open the project folder in VS Code
2. Open the terminal
3. Run the program using:

```bash
python todo.py
```

or

```bash
python3 todo.py
```

## Menu Options

| Menu | Function                 |
| ---- | ------------------------ |
| 1    | Tambah task              |
| 2    | Lihat semua task         |
| 3    | Tandai sebagai selesai   |
| 4    | Tandai sebagai prioritas |
| 5    | Hapus task               |
| 6    | Keluar                   |

## Program Explanation

### tambah_task()

Used to add a new task into the task list.

### lihat_task()

Displays all existing tasks and their status.

### tandai_selesai()

Marks a selected task as completed.

### tandai_prioritas()

Moves a selected task to the top of the list and marks it as priority.

### hapus_task()

Deletes a selected task from the list.
